Weather-app
A weather app that fetches the current weather from the OpenWeather API. The user can search for more than 220 cities around the world.

Features
City Search: Users can search for weather information in more than 220 cities.
Temperature Unit : Users can veiw temparature in Celsius.
Responsive Design: The app works well on different devices.
